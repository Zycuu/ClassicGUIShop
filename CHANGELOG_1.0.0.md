# ClassicGUIShop 1.0.0 Release Candidate

- Colored Blocks now opens a block-type menu before showing individual colors.
- Added Wool, Carpet, Concrete, Concrete Powder, Terracotta, Glazed Terracotta, Stained Glass, Stained Glass Panes, Beds, Candles, Banners, Shulker Boxes, and Other Colored Blocks subcategories.
- Buy and sell modes retain the same colored-block navigation structure.
- Configuration `_notes` now explains every setting or listing field and what changing it affects.
- Removed unnecessary behavioral commentary from enchantment config notes.
- Updated the final `_about` message to Zycu's Bukkit-era ShopGUI message.
- Preserved compatibility with existing 0.5.0 settings, shops, enchantments, balances, and player records.
