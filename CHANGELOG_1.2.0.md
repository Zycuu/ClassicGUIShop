# ClassicGUIShop 1.2.0

- Added `/adminshop edit`, `/adminshop gui-edit`, and `/adminshop editor`.
- Added a visual category and listing browser that reveals every listing, including items with buy and sell set to 0.
- Added exact buy and sell price entry through vanilla anvil text prompts.
- Added persistent custom folders for normal shop categories.
- Added folder creation, renaming, deletion, and item assignment from the editor.
- Added folder-aware player shop navigation similar to the colored-block type menus.
- Added `config/guishop/folders.json` so folder organization stays separate from pricing and exact ItemStack data.
- Added the `guishop.admin.editor` permission with OP level 2 as the default fallback.
- Release artifact: `ClassicGUIShop-26.1.1-v1.2.0.jar`.
