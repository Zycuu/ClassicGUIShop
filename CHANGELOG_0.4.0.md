# ClassicGUIShop 0.4.0

- Enchantment purchases now provide enchanted books instead of modifying held equipment.
- Shop listings can preserve enchantments, custom names, lore, trims, potion contents, and other item components.
- Buying and selling component-rich listings uses exact item matching.
- Vanilla obtainable items are synchronized into vanilla-style default categories.
- Spawn eggs, operator-only blocks, technical blocks, and known unobtainable items are excluded from automatic population.
- Removing a category now removes every listing inside it.
- Removed default categories remain disabled until recreated by an administrator.
- Added `/adminshop catalog sync`.
